# B站热门视频数据分析可视化系统

#### Description
本系统采用Java技术SpringBoot框架和Python技术Django框架（两种框架开发的都有）和Hadoop，hdfs，Scrapy爬虫，MySQL数据库，B/S 结构，Vue.js 技术，算法亮点，采用随机森林回归算法进行预测和协同过滤算法(余弦相似性）推荐。
